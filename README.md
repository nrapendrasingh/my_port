# ✨ Modern 3D Portfolio

A highly interactive and visually stunning 3D portfolio website built with React, Vite, Tailwind CSS, and Three.js. This portfolio showcases my skills as a Cyber Security Analyst and Software Developer, featuring seamless animations, an immersive 3D canvas experience, and fully responsive design.

## 🚀 Features

- **Immersive 3D Experience**: Uses Three.js and React Three Fiber to render interactive 3D models (Computer and Earth) directly in the browser.
- **Sleek Animations**: Powered by Framer Motion for smooth scroll effects, hover states, and dynamic elements.
- **Fully Responsive**: Optimised for all devices, from mobile phones to high-resolution desktop screens.
- **Dynamic Projects**: Display your projects with tilt animations, tags, and quick links to live sites and source code.
- **Live Contact Form**: Integrated with EmailJS, allowing visitors to send messages directly from the website without needing a backend server.
- **Dark Mode Default**: A modern, dark-themed aesthetic with cyber-security inspired accents.

## 🛠️ Technology Stack

- **Frontend Framework**: React 19 + TypeScript
- **Build Tool**: Vite 8
- **3D Graphics**: Three.js, React Three Fiber, React Three Drei
- **Styling**: Tailwind CSS, Vanilla CSS
- **Animations**: Framer Motion
- **Tooling**: Node.js, Biome (Linting/Formatting)

## 💻 Running Locally

To run this project on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/nrapendrasingh/my_port.git
   cd my_port
   ```

2. **Install dependencies:**
   Make sure you have Node.js and `npm` or `bun` installed.
   ```bash
   npm install
   # or
   bun install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   # or
   bun run dev
   ```

4. **Open in Browser:**
   Navigate to `http://localhost:5173/` to view the website.

## 📦 Building for Production

To create a production-ready build of the portfolio:

```bash
npm run build
```
This command compiles the TypeScript code, bundles everything with Vite, and generates the `dist/` directory, which can be easily deployed to Vercel, Netlify, or any other hosting provider.
